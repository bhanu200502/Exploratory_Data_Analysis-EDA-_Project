# Exploratory Data Analysis (EDA) on Titanic Dataset

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) of the Titanic dataset to uncover patterns, trends, and factors influencing passenger survival. Various statistical summaries and visualizations were used to analyze the dataset and generate meaningful insights.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## EDA Process

### Data Exploration

* Dataset loading and inspection
* Shape and structure analysis
* Statistical summary generation

### Data Cleaning

* Missing value identification
* Age column imputation using median
* Embarked column imputation using mode
* Cabin column removal due to excessive missing values

### Visual Analysis

* Age Distribution
* Gender Distribution
* Passenger Class Distribution
* Survival Analysis
* Correlation Heatmap
* Outlier Detection
* Age vs Fare Relationship Analysis

## Key Insights

1. Most passengers belonged to the age group of 20–40 years.
2. Male passengers were more numerous than female passengers.
3. Third-class passengers represented the largest passenger group.
4. Female passengers had a significantly higher survival rate.
5. First-class passengers had better survival chances compared to other classes.
6. Fare contained several high-value outliers.
7. Passenger class strongly influenced survival outcomes.
8. Age showed only a weak relationship with survival.

## Outcome

This project improved understanding of exploratory data analysis, statistical interpretation, visualization techniques, and data-driven decision making.
